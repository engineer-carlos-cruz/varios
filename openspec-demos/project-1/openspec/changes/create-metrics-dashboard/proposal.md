## Why

El proyecto requiere una pantalla principal de dashboard de métricas para visualizar el rendimiento de un producto digital. Esta funcionalidad es fundamental para que los usuarios puedan monitorear KPIs esenciales de forma rápida y clara.

## What Changes

- Crear pantalla única de dashboard con header, métricas, gráfico y tabla
- Implementar selector de rango de fechas que actualiza los datos
- Mostrar estados de loading y vacío para mejor UX
- Usar datos mockeados para todas las métricas

## Capabilities

- `metrics-dashboard`: Pantalla principal que muestra métricas clave, gráfico de tendencia y tabla de actividad reciente
